Projects

Automatic Navigation of a Microrobotic Swarm for Tracking Multiple Mobile Targets

- Propose an optimized rapidly-exploring random tree star (ORRT*) algorithm to generate desired paths connecting the swarm and the targets.

- Develop a path selection strategy to select the shortest path.

- Present a sliding mode (SM) motion controller to achieve the tracking of multiple mobile targets using microswarms.

Chinese University of Hong Kong, Shenzhen

Dynamic Path Planning and Motion Control of Microrobotic Swarms for Mobile Target Tracking

- Present an enhanced bidirectional rapidly-exploring random tree star (EB-RRT*) algorithm considering the physical size of the swarm to dynamically plan the optimal path for obstacle avoidance.

- Propose an image-guided motion controller, which consists of a direction controller and a Genetic Algorithm based Linear Quadratic Regulator (GA-LQR) velocity controller, to realize mobile target tracking using microrobotic swarms.

Chinese University of Hong Kong, Shenzhen

Investigating the magnetic properties of Fe-Co alloys produced by metal injection molding(MIM)

- Using different sintering technologies to control the contents of impurities, such as C and O, in the sintered material, which in turn tailors the magnetic properties of the materials.

Southern University of Science and Technology
